# pacai-cli

A simple CLI tool for creating projects from GitHub templates.

## Installation

```bash
npm install -g pacai-cli
```

## Usage

```bash
# Create a new project
zr create <app-name>

# Force overwrite if directory exists
zr create <app-name> --force

# Show help
zr --help

# Show version
zr --version
```

## Features

- Interactive template selection from your GitHub repositories
- Version/tag selection
- Beautiful loading animations
- Automatic error handling and cleanup
- Colorful console output
